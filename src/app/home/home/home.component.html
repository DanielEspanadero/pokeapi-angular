Introducción

Para comenzar un nuevo proyecto con Angular, primero deberás instalar Node.js y npm, que son dos herramientas fundamentales para el desarrollo de aplicaciones web con este framework. Una vez instalado Node.js y npm, puedes instalar Angular CLI, que es una herramienta de línea de comandos que te permite crear y administrar proyectos de Angular de forma sencilla. Para instalar Angular CLI, abre una terminal y ejecuta el siguiente comando: 

    

        npm install -g @angular/cli

Una vez instalado Angular CLI, puedes crear un nuevo proyecto de Angular ejecutando el siguiente comando en la terminal (reemplaza mi-proyecto con el nombre que desees para tu proyecto):

    ng new nombre-proyecto

Te preguntará una serie de preguntas sobre el estilo de hoja y si quieres añadir el enrutamiento, puedes seleccionar las opciones que prefieras. Después de crear el proyecto, puedes entrar en su directorio y luego puedes iniciar el servidor de desarrollo local y abrir la aplicación en tu navegador con:

    ng serve

Después de ejecutar este comando, abre tu navegador y navega a http://localhost:4200/ para ver tu nueva aplicación de Angular en acción. Tambien puedes ejecutar el siguiente comando para que se abra automaticamente el navegador: 

    ng serve -o
